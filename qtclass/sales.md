# 量潮课堂销售指南

## 为什么要单独写销售

和教育机构的传统招生不同，我们的销售不是"学生来咨询→我们介绍课程→学生报名"这条单向路径。

我们的学生画像很特殊——他们优质但小众。如果不精准触达、不按销售转化的逻辑来，光靠课程介绍是转不动的。

本文档要回答三个问题：
1. 我们的学生画像是谁
2. 怎么找到这些画像的人
3. 怎么完成从触达到转化的全流程

## 学生画像

### 核心特征

**优质：**
- 在学习上有付费习惯——愿意为提升自己花钱
- 接触过的大部分课程或服务质量都不好——有比较才有鉴别
- 对比之下能感受到我们服务质量高
- 主动性强——会自己找资源、自己提问、自己跟进

**但小众：**
- 这类学生在主流市场上并不常见
- 大部分人没有付费学习的习惯，或者只愿意为"证书/学历"付费
- 愿意为"能力提升"付费、且能识别质量差异的人，本身就是一个筛选过的群体

### 一句话画像

> 在学习上花过钱、踩过坑、能识别好坏、愿意主动学的人。

### 反画像（什么人不是我们的目标）

- 只问价格不问内容的人
- 等着被喂、不主动参与的人
- 认为教育应该免费、对付费有抵触的人
- 为"拿证"而不是"学会"来的人

## 怎么找到他们

因为群体小众，广撒网效率很低。需要定向触达。

### 渠道优先级

| 优先级 | 渠道 | 理由 |
|--------|------|------|
| 1 | 现有学员推荐/转介绍 | 画像最准，信任成本最低 |
| 2 | 高校合作渠道（如浙理工） | 有学校背书，批量触达 |
| 3 | 技术社区/开源社区 | 主动学习者聚集地 |
| 4 | 就业焦虑场景（招聘平台、求职群） | 痛点明确，转化路径短 |
| 5 | 大众广告/信息流 | 效率最低，不推荐早期使用 |

### 判断是否是对的人

不要让销售在"这个人是不是目标用户"上靠感觉。用几个快筛问题：

1. 你之前为学习花过钱吗？花在什么地方？
2. 你觉得之前那些课程/服务怎么样？满意吗？为什么不满意？
3. 你这次想学，是想要一个证书，还是想真的掌握这个能力？
4. 你每周能花多少时间在学习上？

前三个问题筛画像，第四个问题筛履约能力。

## 从触达到转化：销售的完整链路

这是我们和当前团队最大认知差的地方。

**"说服用户上课"和"完成销售转化"是两件事。**

| | 说服上课 | 销售转化 |
|--|---------|---------|
| 目标 | 让用户觉得课程不错 | 让用户完成下单→上课→完课的完整闭环 |
| 结束标志 | 用户说"好的我考虑一下" | 用户付款、上完第一节课、续费/推荐 |
| 责任范围 | 介绍课程 | 覆盖售前→售中→售后全流程 |

### 转化漏斗

```
触达 → 建立信任 → 确认画像 → 展示匹配价值 → 促成下单 → 履约跟进 → 续费/转介绍
```

每一层都有明确的动作和判断标准：

#### 1. 触达
动作：通过上面渠道让学生知道我们存在。
判断标准：对方愿意继续聊，而不是已读不回。

#### 2. 建立信任
动作：展示真实案例（浙理工合作成果、往期学员作业/反馈）、开放试听/体验课。
判断标准：对方开始主动提问，而不是被动应答。

#### 3. 确认画像
动作：用上面的快筛问题判断是否是目标用户。
判断标准：符合画像 → 继续推进。不符合 → 不浪费精力，推荐其他更合适的资源。

#### 4. 展示匹配价值
动作：针对对方的痛点（"之前上的课质量差""学完找不到工作"），具体说明我们的课程怎么解决。
关键：不要背参数（"我们有 XX 课时""XX 位老师"），要说结果（"上一期学员结业后 X 周内拿到了面试"）。

#### 5. 促成下单
动作：明确报价、服务内容、交付时间表。消除顾虑（担心学不会 → 有助教跟进；担心没时间 → 弹性学习安排）。
判断标准：对方付款。

#### 6. 履约跟进
动作：开课提醒、学习进度跟踪、问题响应。
目的：确保用户上完课，而不是付了钱就消失。完课率直接影响续费和口碑。

#### 7. 续费/转介绍
动作：课程结束时主动询问体验反馈，引导推荐。
判断标准：用户愿意推荐给别人。

### 常见错误

| 错误 | 表现 | 纠正 |
|------|------|------|
| 撒网不筛选 | 见人就推，不管是不是目标 | 先筛画像，再花时间 |
| 只介绍不转化 | 讲完课程就等对方主动 | 每个接触点都要有"下一步" |
| 成交即结束 | 付款后就不管了 | 履约跟进和售前一样重要 |
| 不敢报价 | 怕报完价人跑了，一直拖着 | 报价是转化的必要环节，越早明确越好 |

## 团队意识建设

销售不是"销售岗位"的事。课程团队每个人都需要建立商业化意识。

### 需要反复强调的三句话

1. **我们是一个商业化的课程团队，不是公益教育组织。** 课程质量好是基础，但质量好不等于自动有人来买。主动做销售转化是每个人的职责。

2. **画像不明确的触达是浪费双方时间。** 跟一个不合适的人聊三天，不如花三天找到两个对的人。

3. **用户付了钱只是开始。** 完课率和满意度决定了这个用户会不会带来下一个用户。履约跟进和售前一样重要。

### 每周复盘问题

- 这周触达了多少人？其中几个人符合画像？
- 转化率多少？哪个环节流失最多？
- 流失的原因是我们的问题（话术、价格、信任）还是画像筛选没做好？
- 有没有哪个用户特别满意的案例可以复用？
