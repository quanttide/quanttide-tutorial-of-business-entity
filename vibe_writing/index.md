# Vibe Writing（氛围写作）

Vibe Writing（氛围写作）是一种利用多个大型语言模型进行迭代式文档创作与优化的方法论。

## 核心理念

不同大型语言模型在文本处理上存在能力差异与"风格偏好"。通过建立结构化的协作流程，让各模型专注于其相对优势环节，可以突破单一模型的思维定式与能力局限，从而系统性提升文档的整体质量。

## 核心工作流

典型的 Vibe Writing 工作流采用"评审-重写"的循环迭代模式：

```
初稿 → 评审模型（GLM） → 评审意见 → 写作模型（DeepSeek）重写 → 定稿
```
