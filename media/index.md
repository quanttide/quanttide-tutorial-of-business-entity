# 以“非评判”的心态运营公司账号

## 评判是表达最大的阻碍

每次打开公司账号看发布数据，我都要做一次心理建设。

不是怕数据差——数据差我其实习惯了。我怕的是看到数据的那一瞬间，脑子里自动蹦出来的那句话：

“又没做好。”

“又没做好”是一个评判。它不告诉我哪里可以改进，不告诉我这次和上次有什么不同，它只是站在那里，像一个面无表情的裁判，举着牌子：不合格。

我发现我所有做得好的内容，都是在“没在看数据”的时候做出来的。写的时候脑子里想的是“这件事真有意思”，而不是“这个标题打开率够不够”。

但平台算法不奖励“有意思”，它奖励“被打开”。于是我被逼着在写完正文之后，再花同样的时间去想一个我自己都不喜欢的标题。

这很荒谬。但我理解平台——它要的是注意力，不是质量。就像番茄小说推那些情绪胶囊式的短篇，连续看几篇就难受，但平台不在乎你难不难受，它只在乎你点开了没。

## 以非评判的心态运营

所以我现在试着做一件事：写完之前，不评判。

不评判数据好不好，不评判这个想法有没有用，不评判读者会不会喜欢。先写完再说。

写完再回头看。那时候评判才有意义——因为东西已经在了，评判它好不好，不影响它存在过。

这很难。我的脑子习惯了边写边打分。但我发现一个规律：每次我忍住不打分，写出来的东西反而有人看。每次我边写边改、边写边焦虑，最后往往发不出来。

所以“非评判”不是躺平，是先把裁判请出场地，让选手先把动作做完。

写到这里，我突然觉得轻松了一点。不是说问题解决了——平台算法还在，标题还是要起。但我找到了一个属于自己的节奏：先空着标题写，先不想数据发，先把困难说出来。
