# 你怎么干 —— 个人视角

如果你是第一天入职，按下面几步走：

## Day 1 待办

- [ ] 确认部门、职级、职务
- [ ] 更换飞书和企业微信头像为证件照
- [ ] 加入飞书工作群（全员群、部门群、必要的工作群）
- [ ] 加入企业微信（商务/销售岗位）
- [ ] 加入 GitHub 组织（技术岗位）
- [ ] 加入小红书群（运营/商务岗位）
- [ ] 浏览飞书知识库
- [ ] 了解工时审计制度和申报操作
- [ ] 确认薪酬结构

## 工时与薪酬

薪酬 = 时薪 × 有效工时。工资每月 15 号发放。

有效工时是产生岗位预定价值的活动。你的直属上级初审，秘书处和 HR 联合终审。做得快加绩效，做得慢减绩效。

建议每天花 5 分钟记录当天工作，每周至少申报一次工时，不要积压到月底。

## 会议与汇报

- 向你的直接上级汇报日常工作
- 部门每周召开部门办公会
- 跨部门事项提交公司代表大会周会审议
- 会议材料需提前 48 小时提交
- 不要越级汇报

## 文档规范

公司制度、模板统一存放在飞书——云文件——知识库。

快速找到文档：Ctrl+K 全局搜索，输入关键词即可。

文档编辑完成后，如果属于公司公共知识，请移动到知识库对应目录下。

## 遇到问题

先查知识库，再看本教程，还有问题找秘书处。
